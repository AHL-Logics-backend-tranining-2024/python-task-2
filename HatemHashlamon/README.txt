This is a console application for task managment.
It has the following features :
1. Add a new task to do
2. View all of your tasks that you have assigned.
3. Update any part of a specific task.
4. Delete a task that you have completed or no longer want to do.


Notes : 
1. Task ids are auto generated, it will be printed once you create the task, and you can always view it using the second option.
2. you have the option of specifing that task as urgent, then you should specify the priority of that urgent task.
2. You should enter the status (and the priority for urgent tasks) exactly as written in order for it to work (will be updated in the future).
